## Project CRUD 

# What is the project?
A simple CRUD application, focused on learning. Connects to MongoDB without any ORM.


### Lenguages and FrameWorks
```
- TypeScript
- Fastify
```
### DataBase
```
-  MongoDB 
```

### About the project

The project focused on understanding how a direct connection between the _database and Fastify works_, without using any **ORM**. 
The project only contains the back-end and has no security checks. 
**The real goal was to understand the pure connection.**
